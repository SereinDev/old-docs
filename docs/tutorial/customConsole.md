
# ✏ 自定义控制台

自定义控制台的背景、字体

:::caution
仅WinForm和v1.3.3及以前的WPF
:::

## 具体步骤

1. 打开`./console`文件夹
2. 在Visual Studio Code或者其他编辑器中编辑其中的文件
3. 重启Serein后生效

## 提示

- console.html 中定义的函数**不建议更改**，修改错误可能导致无法输出信息
- 由于显示网页组件默认以IE为内核，修改时内嵌网页时需注意**适配IE浏览器**
