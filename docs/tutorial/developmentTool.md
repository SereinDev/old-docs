# 🔌 开发工具选项

## 开启方法

打开 settings/Serein.json 按需修改以下配置项

```json
{
    // ...
    "DevelopmentTool": {
        // highlight-next-line
        "EnableDebug": true,            // 开启调试模式
        // highlight-next-line
        "DetailDebug": false,           // 详细的调试输出（包含函数参数类型和返回值，仅当上一行启用时生效）
        "NOTE": "以上设置内容为开发专用选项，请在指导下修改"
    }
}
```

:::danger
**如果你不知道这些会产生什么效果，请勿随意修改**
:::
